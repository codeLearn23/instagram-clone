# Instagram Clone

## Overview

This is an Instagram UI imitation that is the result of a solo project, "Instagram Clone", in the [Fullstack](https://scrimba.com/courses) curriculum at Scrimba. The [style source](https://www.figma.com/design/h0MKma9TTWzGOMQ9Ia6ROW/Oldagram?node-id=0-1&p=f) is located on Figma.

## Things learned

- how to improve accessibility within a program
- how to render an array of objects to the page with a loop and have styles apply to each object
- how to build the HTML elements and content within JavaScript, and then append them to a root div
